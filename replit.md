# Monsaik

## Overview
Starter project imported from GitHub (Bimon420/monsaik). The original repository contained only a placeholder readme, so a minimal static web page was created as a starting point.

## Structure
- `index.html` — Main landing page
- `server.py` — Simple Python HTTP server serving files on port 5000

## Running
The "Start application" workflow runs `python server.py` which serves the project on port 5000.

## Deployment
Configured as a static site deployment.
